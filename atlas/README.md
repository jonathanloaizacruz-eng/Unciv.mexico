# Atlas de Imágenes para México

Para completar el mod, reemplaza los archivos placeholder con imágenes PNG reales de 256x256 píxeles. Descarga de fuentes libres como Pixabay.com, Unsplash.com o crea tus propias.

Sugerencias:
- Unidades: Busca "guerrero azteca", "mariachi", etc.
- Edificios: Busca imágenes de los monumentos reales (Templo Mayor, Pirámide de Kukulcán, etc.).

Asegúrate de que los nombres coincidan con los del archivo MexicoAtlas.atlas.

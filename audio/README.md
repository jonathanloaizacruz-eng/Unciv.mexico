# Sonidos de México

Para completar el mod, reemplaza los archivos placeholder con sonidos OGG reales. Descarga de fuentes libres como Freesound.org o Zapsplat.com.

Sugerencias:
- nation_music.ogg: Música mariachi o folklórica mexicana (ej. https://freesound.org/people/klankbeeld/sounds/191020/)
- leader_music.ogg: Música épica para Porfirio Díaz.
- attack_sound.ogg: Grito de guerra azteca o disparo.
- defeat_sound.ogg: Música triste.
- victory_sound.ogg: Celebración con mariachi.

Convierte a OGG si es necesario usando herramientas como ffmpeg.
